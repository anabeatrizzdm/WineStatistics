# Análise estatística de vinhos usando o dataset do Kaggle 

Este é um projeto de análise estatística de um dataset de vinhos, realizado como parte de um projeto de ciência de dados. O dataset utilizado contém informações sobre vinhos, incluindo pontuações, preços, variedades, países de origem, entre outros.

## Objetivo

O objetivo deste projeto é realizar uma análise descritiva dos dados para extrair insights sobre os vinhos e as características que podem influenciar suas pontuações e preços. Além disso, busca-se explorar a relação entre a pontuação dos vinhos e outras variáveis, como país de origem, variedade de uva e vinícola produtora.

## Conteúdo

O dataset utilizado neste projeto contém as seguintes colunas:

- `id`: Identificador único do vinho
- `country`: País de origem do vinho
- `points`: Pontuação atribuída ao vinho
- `price`: Preço do vinho
- `province`: Província de origem do vinho
- `region_1`, `region_2`: Região de origem do vinho
- `title`: Título da avaliação do vinho
- `variety`: Variedade de uva do vinho
- `winery`: Vinícola produtora do vinho

## Análises Realizadas

1. **Média, Mediana e Desvio Padrão das Pontuações dos Vinhos**: Foram calculadas a média, mediana e desvio padrão das pontuações dos vinhos.
2. **Comparação das Pontuações entre Diferentes Países**: Foi realizada uma análise das pontuações dos vinhos em diferentes países.
3. **Comparação de Pontuações entre Diferentes Variedades de Uva**: Foi analisada a variação das pontuações entre as diferentes variedades de uva.
4. **Comparação de Pontuações entre Diferentes Vinícolas**: Foi analisada a variação das pontuações entre as diferentes vinícolas produtoras.
5. **Correlação entre Preço e Pontuação**: Foi calculada a correlação entre o preço e a pontuação dos vinhos.
6. **Estatísticas Descritivas dos Preços**: Foram calculadas a média, mediana e desvio padrão dos preços dos vinhos.
7. **Histograma da Distribuição dos Preços**: Foi criado um histograma para visualizar a distribuição dos preços dos vinhos.
8. **Frequência de Vinhos de Cada País**: Foi analisada a frequência de vinhos de cada país no dataset.
9. **Boxplot dos Pontos dos Vinhos**: Foi criado um boxplot para visualizar a distribuição dos pontos dos vinhos.
10. **Top 15 Variedades de Uva Mais Comuns**: Foi analisada a frequência das top 15 variedades de uva no dataset.
11. **Contagem das Top 10 Variedades de Uva por País**: Foi criado um gráfico de barras para visualizar a contagem das top 10 variedades de uva por país.
12. **Pontuação Média por Variedade de Uva**: Foi criado um gráfico de barras para visualizar a pontuação média das top 10 variedades de uva.
13. **Pontuação Média por Título de Vinho**: Foi criado um gráfico de barras para visualizar a pontuação média dos top 6 títulos de vinho.
